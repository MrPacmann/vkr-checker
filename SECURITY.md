# Безопасность и конфиденциальность

Приложение предназначено для локальной предварительной проверки учебных документов. Загружаемые `.docx` и `.pdf` обрабатываются в браузере пользователя и не отправляются на сервер приложения.

В проекте не должно быть учебных работ, отчётов, персональных данных студентов, экспортированных HTML-отчётов и иных конфиденциальных материалов. Перед публикацией на GitHub проверьте, что такие файлы не добавлены в репозиторий и не попадают в коммиты.

Если вы обнаружили в истории репозитория конфиденциальный документ, удалите его из истории Git до публикации и при необходимости перевыпустите репозиторий.
